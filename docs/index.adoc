=== Build

Build sources:

----
./gradlew build
----

Deploy to maven local:

----
./gradlew publishToMavenLocal
----

Build the documentation page:

----
mvn verify
----

Upload the documentation page on GitHub pages:

----
mvn exec:exec
----
